**Hollow Knight Showcase**
Interactive 3D website built with Three.js, featuring animated character models from Hollow Knight with dynamic lighting and mouse based controls. Each character has its own immersive scene and description inside a dark, game inspired UI.
Built with:
HTML, CSS, JavaScript, Three.js
**Features:**
real-time 3D models, interactive rotation, atmospheric lighting.
